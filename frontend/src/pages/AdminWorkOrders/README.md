# AdminWorkOrdersPage Submodules

This folder contains extracted logic and components for the AdminWorkOrdersPage to improve scalability and maintainability.

- `colorSchemes.ts`: Color scheme logic and helpers
- `schemas.ts`: Zod schemas and form types
- `dndBoard.tsx`: DnD/board logic and helpers
- `FilterBar.tsx`: Filter bar and filter logic
- `AdminWorkOrdersPage.module.css`: CSS module for custom styles
